DESCRIÇÃO DO PROJETO : Esse projeto foi desenvolvido por uma aplicação backend utilizando Java e spring Boot para gerenciar o cadastro de beneficiários de um plano de saúde por meio de uma API RESTful; No qual permite fazer as operações básicas de CRUD (Criar, Ler, Atualizar e Deletar) relacionadas aos beneficiários e seus documentos.

TECNOLOGIA UTILIZADAS : H2 Database, Lombok, Spring Boot, Spring Data JPA...

A aplicação estará acessível via http://localhost:8080.

A documentação dos Endpoints através do postman.
